# STM32L4KC-Development-Board
A custom made mini development board for the STM32L432KC6 low power MCU.

## Features
* STM32L432KCU6 MCU
* Serial Wire Programming Headers
* USB-C Power Connection
* 8mhz external crystal for precise timing
* Boot and reset switches
* Breakout of 21 GPIO pins

## Schematic and Layout
A pdf of the schematic can be viewed here [Schematic.pdf](https://github.com/user-attachments/files/29484714/Schematic.pdf) and kicad project files are included as well for your own alterations.
I tried my best to keep all components on one side of the board and the footprint as small as possible, and in a form that can easily be inserted into a breadboard much like an arduino nano or stm nucleo with an L4 chip like this one.
